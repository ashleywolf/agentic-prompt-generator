# Reporting Format Guide

Import this file for consistent report formatting across workflows.

## Standard Report Sections

### Header
```markdown
## [emoji] [Report Title] — [date or date range]
```

### Metrics Block
```markdown
| Metric | Value | Change |
|--------|-------|--------|
| [name] | [value] | [trend emoji] [delta] |
```

### Item List
```markdown
### [Section Title]
- **[item title]** ([#number](link)) — [brief description]
  - Labels: `label1`, `label2`
  - Author: @username
```

### Summary Footer
```markdown
---
*Generated by [workflow name] on [timestamp]. [link to workflow run]*
```
